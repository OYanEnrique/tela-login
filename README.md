# Login Screen

Este é um exercício do módulo 4 do curso de HTML5 e CSS3 do Curso em Vídeo, ministrado pelo professor Gustavo Guanabara.

## Descrição

Projeto de uma tela de login desenvolvida como parte do aprendizado de HTML5 e CSS3.

## Estrutura do Projeto

```
login-screen/
├── index.html
├── estilos/
│   └── styles.css
└── imagens/
```

## Tecnologias Utilizadas

- HTML5
- CSS3

## Como Usar

Abra o arquivo `index.html` em seu navegador para visualizar a tela de login.
